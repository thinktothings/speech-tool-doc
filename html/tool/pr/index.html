第一步:srt直接导入PR(乱码演示)

  • srt格式直接导入PR会以乱码显示
  • 需要对应格式的文件编辑才可以显示
  • 每个软件默认的文件编码有区别、就如window默认编码是GBK,Mac系统默认编辑是UTF-8

第二步:免费文本编辑工具转码(支持PR)

  • 本地需安装免费文本编辑软件:Notepad++
  • 选中srt文件 ==> 右键选中(按钮) Edit with Notepad++
  • 工具栏 == 编码 ==> 使用UTF-8-BOM 编码
  • 文件 ==> 保存
  • 些时导入PR就不会有乱码问题了
